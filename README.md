# 💬 Real-Time Chat Application

A full-stack real-time chat application built using the MERN Stack and Socket.IO. The application enables users to communicate instantly with secure authentication, online user status, and a responsive user interface.

---

## 🚀 Features

- 🔐 User Authentication (JWT)
- 💬 Real-Time One-to-One Messaging
- 🟢 Online/Offline User Status
- 📸 Profile Image Upload
- 🔒 Secure HTTP-Only Cookies
- 📱 Fully Responsive Design
- ⚡ Instant Message Delivery using Socket.IO
- 🎨 Modern UI with Tailwind CSS
- 🔄 State Management using Redux Toolkit

---

### Frontend
